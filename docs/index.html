股票教练智能体使用说明

stock-trading-coach-agent 用于每日交易复盘：交易事实、交易想法、文章观点、宏观镜片、叙事污染检查、行为诊断和长期 playbook。

它做什么

保留交割单本地脱敏、隐私检查、指标计算和行为诊断。
记录 daily journal：当天交易想法、意图、情绪和计划。
提取文章观点并检查叙事污染。
手动蒸馏宏观博主观点，形成市场环境观察镜片。
生成每日 HTML、Markdown 和 JSON 教练报告。
生成可发雪球的每日复盘与明日计划草稿。
保守沉淀 playbooks.json：可复制、待验证、应避免。
基于历史亏损模式生成买入前风控反问。

它不做什么

  • 不荐股。
  • 不预测未来涨跌。
  • 不提供买入、卖出或持有某只股票的建议。
  • 不把真实 PDF、CSV、XLSX 或截图提交到 GitHub。

默认用法

在 Codex 中说：

请使用 $stock-trading-coach-agent

或手动运行：

python3 scripts/interactive_runner.py --open

打开 http://127.0.0.1:8787 后上传交易文件，填写交易想法和文章信息。最终报告在 local_outputs/run_时间戳/daily_coach_report.html，稳定入口为 local_outputs/daily_coach_report.html。

雪球发布版在 local_outputs/run_时间戳/daily_xueqiu_post.html，稳定入口为 local_outputs/daily_xueqiu_post.html。

手动更新宏观镜片

例如蒸馏冰冰小美雪球主页最近文章：

python3 scripts/macro_lens_digest.py --source xueqiu --user-url "https://xueqiu.com/u/7143769715" --limit 50
  • 只保存标题、URL、摘要、宏观镜片和风险标签，不保存全文。
  • 宏观镜片只用于市场情况判断、教练解释和明日条件计划。
  • 如果雪球页面需要登录或反爬，脚本会提示使用具体文章 URL 或粘贴文本。

隐私边界

真实文件只在本机处理。PDF 原文不交给 AI。截图成交单只有在你显式发给 Codex 识别时才由 AI 提取标准 trades；请先裁剪或打码姓名、账号、资金余额。

  • 原始上传文件只进入系统临时目录，处理后删除。
  • local_outputs/ 保存真实报告，local_state/ 保存长期 playbook；两者都不提交 Git。
  • 文章 URL 可联网抓取，但长期只保存标题、URL、观点摘要和污染检查，不保存全文。
  • 雪球发布版可显示股票代码和名称，但成交金额默认转换为“单位”表达。

Playbook 规则

  • 单次交易成功不得直接进入“可复制”。
  • 至少 3 次类似证据后，才能从“待验证”升级为“可复制”。
  • 亏损或风险失控模式进入“应避免”。
  • 每条模式必须包含触发条件、入场理由类型、退出方式、最大风险、证据日期、验证状态。
