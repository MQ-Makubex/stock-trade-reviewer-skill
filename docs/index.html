股票交割单 AI 复盘 Skill 使用说明

这是一个面向 Codex 的中文 Skill，用于在本机清洗券商交割单、计算历史交易指标，并生成个人交易复盘报告。

它做什么

从 CSV、Excel 或文字型 PDF 交割单中提取成交记录。
在本机脱敏、检查隐私字段、标准化交易数据。
计算盈亏、费用、单票表现、交易生命周期和行为模式。
生成 Markdown 和 HTML 个人交易复盘报告。

它不做什么

  • 不荐股。
  • 不预测未来涨跌。
  • 不提供买入、卖出或持有某只股票的建议。
  • 不把真实交割单 PDF 或原始交易数据上传到服务器。

隐私边界

真实 PDF 不要上传给 AI。真实文件只在本机终端运行脚本。运行前请删除姓名、身份证、手机号、资金账号、银行卡号、客户号、股东账号、营业部、地址等信息。

PDF 流程只输出脱敏后的交易字段。资金余额默认删除；如果你主动传入 --keep-balance，仍建议再运行隐私检查。

推荐使用流程

  1. 准备券商交割单 PDF。
  2. 本地脱敏：
python3 scripts/sanitize_pdf_statement.py 交割单.pdf -o sanitized_trades.csv
  1. 隐私检查：
python3 scripts/privacy_guard.py sanitized_trades.csv
  1. 标准化解析：
python3 scripts/parse_statement.py sanitized_trades.csv
  1. 计算指标：
python3 scripts/compute_metrics.py cleaned_trades.csv
  1. 补齐生命周期、行为诊断和反事实模拟：
python3 scripts/build_trade_lifecycle.py cleaned_trades.csv
python3 scripts/detect_behavior_patterns.py cleaned_trades.csv metrics.json trade_lifecycle.json
python3 scripts/counterfactual_simulator.py metrics.json trade_lifecycle.json
  1. 生成复盘报告：
python3 scripts/generate_review_report.py
python3 scripts/generate_html_report.py
  1. 查看 HTML 报告：
open trade_review_report.html

先用虚构样例试跑

项目内的 sample_trades.csv 和 sample_outputs/ 是完全虚构数据，可用于确认环境是否正常。

python3 scripts/parse_statement.py sample_trades.csv -o sample_outputs/cleaned_trades.csv
